# 信息调研类
## 1. 出入境与证件

这是出国旅行的门槛，必须在出发前彻底搞清楚。

### 1.1 签证类型与要求

**免签国家**：

- 需要确认停留天数限制（如日本15天、新加坡30天、马来西亚30天）
- 部分免签有附加条件（如需酒店预订单、回程机票）
- 注意是否需要填入境卡、是否需要返程机票证明

**落地签**：

- 需要准备材料：护照、照片、机票、酒店预订单、签证费（现金）
- 排队时间可能较长，尤其是热门时段
- 部分国家支持电子落地签（e-VOA），可提前网上申请

**电子签（e-Visa）**：

- 需要准备的材料：护照扫描件、照片、资金证明、行程单
- 审批时间从几小时到几天不等，留足提前量
- 打印电子签证确认函，随身携带

**贴纸签/普通签证**：

- 材料准备较多：护照、照片、签证申请表、在职证明、银行流水、保险单
- 送签后等待时间可能较长（美国1-2周，申根3-5个工作日）
- 部分国家需要面试（美国），提前准备好问答准备

### 1.2 护照有效期与空白页

**有效期要求**：

- 大多数国家要求护照有效期在返回日期后6个月以上
- 部分国家要求3个月以上（如马来西亚）
- 如果护照即将过期（不足6个月），尽快去出入境管理局更换

**空白页要求**：

- 签证页一般需要2页以上空白（部分需要4页）
- 注意：加注页不算空白页
- 如果护照快满了，提前申请更换

### 1.3 停留天数与延期

- 确认单次最长停留天数（如泰国60天、马来西亚30天）
- 了解是否可以延期、延期流程和费用
- 注意边境盖章是否清晰，有时会有入境章缺失的问题

## 2. 安全与健康

安全和健康是旅行的底线，需要认真对待。

### 2.1 目的地治安评估

**常见骗术与防范**：

| 类型 | 常见场景 | 防范方法 |
|------|----------|----------|
| 出租车宰客 | 机场/景点外 | 用正规打车软件或提前了解价格 |
| 餐厅宰客 | 旅游区餐厅 | 提前看点评，确认菜单有价格 |
| 假导游 | 景点门口 | 不要理睬主动搭讪的“导游” |
| 汇率陷阱 | 货币兑换点 | 确认汇率和手续费，计算实际到账 |
| 假警察 | 街头 | 警察不会在路上查钱，要求去警局 |
| 换钱诈骗 | 暗巷小店 | 只在正规兑换点或银行换钱 |

**夜间安全**：

- 了解目的地哪些区域夜间不宜前往
- 独自外出避免走偏僻小路
- 保持手机有电，预设紧急联络人

### 2.2 疫苗与健康证明

**需要接种的疫苗**：

- 黄热病：去非洲、南美洲部分国家需要
- 霍乱：部分高风险地区
- 伤寒：日本、东南亚部分地区建议
- 新冠疫苗：部分国家仍要求提供接种证明

**《国际预防接种证书》（小黄本）**：

- 在各地国际旅行卫生保健中心办理
- 需要提供接种记录或现场接种
- 提前了解需要接种的疫苗种类，有些需要分多次接种
- 小黄本有效期：霍乱4年，黄热病终身

### 2.3 食品与饮水安全

**饮水安全**：

- 发达国家：自来水可直接饮用
- 发展中国家：建议购买瓶装水，不喝生水
- 部分地区（东南亚）冰块也可能不干净

**食品安全**：

- 街边小吃选择人多的摊位，降低风险
- 海鲜类确保新鲜，生的贝类谨慎食用
- 对当地食材不熟悉时，避免生食

**常用药品准备**：

- 肠胃药（黄连素、蒙脱石散、益生菌）
- 抗过敏药（氯雷他定）
- 消炎药（头孢类，看情况备用）
- 创可贴、碘伏棉签

## 3. 最佳旅行时间与天气

天气直接影响旅行体验，需要提前做功课。

### 3.1 历史气候数据查询

**需要了解的信息**：

- 月平均气温（日间/夜间）
- 降雨概率和降雨量
- 湿度水平
- 日照时长
- 风速（海边/高山地区需关注）

**查询渠道**：

- Wikipedia目的地页面
- Weather Atlas等气候网站
- 穷游锦囊目的地区天气章节
- 小红书/马蜂窝上近期游客的反馈

### 3.2 季节性风险评估

**雨季考量**：

- 雨天影响户外活动和拍照
- 可能导致道路泥泞、景点关闭
- 但也可能带来独特体验（雨后的寺庙、瀑布水量更大）
- 雨季出行需备好雨具，关注天气预报

**极端气候窗口**：

- 台风季：太平洋地区6-11月，关注台风路径
- 雾季：某些山区可能雾气很重，影响观景
- 高温季：南亚、中东夏季可能超过40度
- 严寒季：北欧、俄罗斯冬季零下30度，需专业防寒装备

### 3.3 穿衣准备建议

| 目的地类型 | 建议衣物 | 配件 |
|-----------|----------|------|
| 热带海岛 | 短袖、短裤、泳衣、薄外套（室内空调冷） | 墨镜、帽子、防晒霜 |
| 温带城市（春秋） | 长袖衬衫、薄外套、牛仔裤 | 折叠伞、轻便外套 |
| 寒冷地区 | 羽绒服、保暖内衣、厚毛衣 | 围巾、手套、暖宝宝 |
| 高山地区 | 冲锋衣、速干层、保暖层 | 登山鞋、墨镜、防风帽 |

**Tips**：

- 洋葱式穿衣法，方便根据温度调整
- 带一件深色外套，耐脏又适合拍照
- 女生尽量带可搭配多套的百搭款，减少行李重量

## 4. 大交通方案

交通是旅行的骨架，选择好的交通方式能省时又省钱。

### 4.1 国际航班选择

**直飞 vs 转机**：

| 类型 | 优点 | 缺点 |
|------|------|------|
| 直飞 | 省时、不折腾、行李直达 | 价格通常较高 |
| 单程转机 | 价格可能更低 | 需中转等候、提取行李重新托运 |
| 两程转机 | 最便宜 | 耗时久、变数多 |

**转机注意事项**：

- 国际转机至少预留3小时（同一航站楼2小时）
- 不同航站楼（如希思罗T1-T5）预留更多时间
- 考虑签证要求：有些国家转机需要过境签

**航司选择**：

- 廉价航空：亚航、捷星、宿雾太平洋——注意行李额和餐食不含
- 全服务航空：国航、东航、全日空——含行李和餐食
- 考虑联盟：寰宇一家、星空联盟、天合联盟，积分可累积

**航班搜索平台**：

- Google Flights：搜索功能强大，价格走势清晰
- Skyscanner：天巡，覆盖广
- 去哪网/携程：中文界面，常见航线价格有时更低

### 4.2 目的地内部交通

**铁路**：

- 日本：新干线四通八达，JR Pass很划算
- 欧洲：Eurail通票适合多国游；提前购票可获折扣
- 东南亚：泰国火车便宜但慢，越南Vietnam Railways
- 国内：高铁覆盖广，提前买票有优惠

**航空**：

- 欧洲内部：瑞安航空、伏林航空等廉价航司
- 东南亚：亚航、狮航国内航线便宜
- 注意廉航行李额限制，提前购买

**巴士**：

- 东南亚：长巴是常见交通方式，建议选VIP车型
- 日本：高速巴士有时比新干线便宜很多
- 欧洲：flixbus覆盖广，价格便宜

**自驾**：

- 需要国际驾照认证件（IDP）或当地认可的翻译公证件
- 租车平台：租租车、惠租车、赫兹（Hertz）、安飞士（Avis）
- 注意：左舵/右舵国家、中国驾照是否被认可

### 4.3 行李额与退改政策

**行李额**：

- 廉航通常不含托运行李，提前官网买比现场便宜
- 国际航班一般包含1-2件23kg托运
- 手提行李：经济舱通常7-10kg，尺寸有限制

**退改政策**：

- 灵活票（Flex Ticket）：可免费改期，差价多退少补
- 标准票：改期收费，特殊情况下可退票
- 廉航票：通常不可退改，或退改费用极高

**建议**：

- 行程不确定时，优先选可退改的票
- 购买旅行保险可覆盖航班延误取消

## 5. 住宿区域选择

住宿位置直接影响旅行体验，便利的位置能省很多时间和精力。

### 5.1 主要住宿区域分析

**城市住宿区域分类**：

| 区域类型 | 特点 | 适合人群 |
|---------|------|----------|
| 交通枢纽附近 | 靠近地铁站、火车站，出行方便 | 第一次去、时间紧张 |
| 老城区/市中心 | 景点集中、夜生活丰富 | 喜欢热闹、想多逛景点 |
| 新城区/商务区 | 设施新、价格适中 | 商务人士、追求安静 |
| 海滨/度假区 | 景色好、节奏慢 | 度假、亲子、蜜月 |

**选择技巧**：

- 用Google Maps测量住宿到地铁站/景点的距离（步行时间）
- 看地图周边有没有便利店、餐厅、药房
- 搜索当地治安较好的区域

### 5.2 住宿类型对比

| 类型 | 优点 | 缺点 | 适合场景 |
|------|------|------|----------|
| 酒店 | 服务稳定、安全、有早餐 | 价格较高 | 首次去、带孩子/老人 |
| 民宿（Airbnb） | 空间大、有厨房、价格灵活 | 卫生参差、可能不合预期 | 多人出行、想体验当地生活 |
| 青旅 | 价格低、能结识背包客 | 私密性差、可能有噪音 | 独自旅行、预算有限 |
| 公寓酒店 | 自助入住、有客厅 | 无人服务 | 长期停留（3天以上） |
| 度假村 | 设施全、一价全包 | 价格高、地点偏 | 海岛度假 |

### 5.3 民宿与酒店的选择标准

**酒店选择要点**：

- 星级只是参考，实际体验看评价
- 重点看位置评分、干净程度评分
- 早餐是否包含（能省不少）
- 是否有停车场（自驾需关注）

**民宿选择要点**：

- 仔细看房屋照片和视频
- 阅读所有评价，重点看差评和房东回复
- 确认入住和退房时间、是否有门禁
- 了解设施：厨房用品、洗衣机、WiFi速度
- 联系房东询问不清楚的问题

### 5.4 阅读差评的技巧

**差评看什么**：

- 最近的差评（1-3个月）最有参考价值
- 重点关注：噪音问题、清洁问题、位置不符、安全问题
- 判断差评是客观问题还是个人预期问题

**好评怎么用**：

- 好评中的具体细节更可信
- 关注重复出现的优点：服务好、位置佳、干净

## 6. 必去与避雷

信息调研的核心目的是找到真正值得去的地方，避免踩坑。

### 6.1 当地人推荐 vs 网红打卡

**获取真实推荐的渠道**：

- Reddit/Quora：搜索目的地相关讨论
- TripAdvisor：看餐厅和景点的本地人评价
- 小众旅行博主：非营销号的真实体验
- 朋友推荐：去过当地的朋友第一手信息

**需要谨慎对待的内容**：

- 过度营销的网红景点
- 没有具体信息的“必去清单”
- 明显是软广的游记
- 时间过久的攻略（可能已过时）

### 6.2 常见踩坑点

**景点类**：

- 游客照骗人的景点（如人工布景、滤镜过度）
- 排队2小时拍照5分钟的地方
- 名不副实的“XX同款XX”
- 过于商业化的老街（千篇一律的义乌商品）

**餐厅类**：

- 只做游客生意的餐厅（菜单有英文中文、价格虚高）
- TripAdvisor上全是5星且评价数很少的
- 没有当地人光顾的“网红店”

**体验类**：

- 质量参差不齐的一日游
- 过度宣传实际很水的项目
- “必买”纪念品（其实淘宝都有）

### 6.3 闭馆日与营业时间

**常见闭馆日**：

- 博物馆：周一闭馆最常见（如巴黎大部分博物馆）
- 教堂：特殊节假日可能关闭
- 餐厅：部分餐厅有固定休息日
- 商店：日本很多店铺周三休息

**季节性调整**：

- 旺季营业时间长，淡季可能缩短
- 冬季部分景点可能关闭维护
- 节假日前后可能有特殊营业安排

**建议**：

- 提前在官网或Google Maps确认开放时间
- 重要景点提前在线预约
- 把闭馆日错开，安排其他活动

## 7. 当地规则与习俗

入乡随俗是旅行者的基本素养，了解当地规则能避免尴尬和麻烦。

### 7.1 小费文化

| 地区 | 小费习惯 | 金额参考 |
|------|----------|----------|
| 美国 | 必须给 | 餐厅15-20%，行李员$1-2/件，导游15-20% |
| 欧洲 | 看情况 | 餐厅10-15%，酒店帮忙$1-2 |
| 日本 | 无需给 | 贴心服务是常态，给小费反而奇怪 |
| 东南亚 | 看情况 | 餐厅10%，导游适当给 |
| 澳洲 | 可选 | 一般不强制 |

**给小费的方式**：

- 餐厅：给现金，刷卡时可以说"keep the change"
- 酒店：放在枕头或信封里
- 导游：行程结束后给

### 7.2 着装礼仪

**宗教场所**：

| 场所 | 要求 |
|------|------|
| 寺庙（泰国、缅甸、柬埔寨） | 覆盖肩膀和膝盖，不可穿短裤短裙 |
| 清真寺 | 女性需要带头巾，男女都需要覆盖手臂和腿 |
| 教堂（梵蒂冈） | 穿着得体，不过度暴露即可 |
| 日本神社 | 无硬性要求，但不要穿太暴露 |

**建议**：

- 带一条围巾，可当披肩也可防晒
- 参观宗教场所前提前了解着装要求
- 如果穿错了，有些地方会提供遮挡衣物

### 7.3 拍照限制

**常见限制**：

- 博物馆内部禁止拍照或禁止闪光灯
- 军事区、机场禁区绝对禁止
- 部分寺庙内部不允许拍照
- 人物摄影需要对方许可（尤其当地人）

**建议**：

- 进入景点前看指示牌
- 不确定就问，不要偷拍
- 拍人之前先征求同意（尤其老人和孩子）

### 7.4 电压与插座

| 地区 | 电压 | 插座类型 |
|------|------|----------|
| 中国 | 220V | 两脚扁插/三脚 |
| 日本 | 100V | 两脚扁插 |
| 美国/加拿大 | 110V | 两脚扁插 |
| 欧洲 | 220V | 两脚圆插 |
| 英国 | 230V | 三脚方插 |
| 澳洲 | 230V | 三脚八字插 |

**准备建议**：

- 携带万能转换插头（最好带USB口）
- 确认充电器支持电压范围（大部分手机充电器100-240V通用）
- 如果有多个设备，带一个小型插线板

## 8. 通讯与网络

网络是现代旅行的生命线，没有网络可能会寸步难行。

### 8.1 电话卡选择

**购买渠道对比**：

| 渠道 | 优点 | 缺点 |
|------|------|------|
| 出发前网上购买 | 有中文客服，到达即用 | 价格稍贵 |
| 机场购买 | 立即可用 | 语言不通，选择有限 |
| 市区营业厅 | 选择多、价格合理 | 需要到市区才能换卡 |

**常见运营商**：

- 东南亚：AIS、True Move、H、Viettel
- 日本：Docomo、Softbank、AU
- 韩国：SKT、KT、LGU+
- 欧洲：各国运营商不同，可买EU漫游卡

**套餐选择**：

- 流量需求：看视频多需要大流量，日常导航查资料10GB足够
- 天数：按行程天数选择，留1-2天余量
- 通话需求：如果需要打电话，选含通话的套餐

### 8.2 多人出行通讯方案

**随身WiFi**：

- 适合4人以下，可同时连接多个设备
- 缺点：需要归还、需要充电、一个设备带所有人都用
- 价格：10-20元/天

**eSIM**：

- 适合有eSIM功能的手机（iPhone XS及以后）
- 优点：不用换卡、可同时用原号码、可多张eSIM
- 缺点：需要手机支持、操作稍复杂

**国际漫游**：

- 优点：最方便，不用换卡
- 缺点：价格高、可能有限速
- 建议：仅作为备用

### 8.3 离线准备

**必须离线的内容**：

- 离线地图（Google Maps离线地图、Maps.me）
- 翻译语言包（Google翻译离线包）
- 酒店确认单PDF
