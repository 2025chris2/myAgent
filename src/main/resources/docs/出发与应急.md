# 出发与应急保障类

> 从出门那一刻起，旅行正式开始。这一模块保障你从容应对突发状况。出发前的准备只是开始，旅途中的应变能力才是关键。

---

## 1. 出发日核对清单

出发日的准备决定了你是否能从容出发。

### 1.1 航班状态确认

**出发当天必做**：

- 出发前3-6小时再次确认航班状态
- 查看是否有延误、取消、登机口变更
- 下载航空公司的App，实时接收推送

**确认渠道**：

- 航空公司官网/App
- 航旅纵横/飞常准等航程App
- 机场大屏（可在出发前2小时查询）

**延误应对预案**：

- 如果延误1-2小时：保持关注，调整机场等候时间
- 如果延误超过3小时：联系航空公司，了解改签可能性
- 如果取消：立即联系改签或退款，准备备选方案

### 1.2 随身物品最后检查

**出门前默念"身手钥钱"**：

| 物品 | 检查方法 |
|------|----------|
| 身份证/护照 | 放在固定位置，伸手可及 |
| 手机 | 电量满格，充电宝满电 |
| 钱包 | 现金+信用卡，备用卡也在 |
| 钥匙 | 家里钥匙+车钥匙（如需要） |

**出门前检查**：

- [ ] 护照在证件包里
- [ ] 手机满电、充电宝满电
- [ ] 钱包里现金和卡够用
- [ ] 家里钥匙在包里
- [ ] 随身行李已检查（无违禁品）

### 1.3 行李优化

**穿脱原则**：

- 把最重/最大的鞋穿身上
- 把最占地方的外套穿身上
- 减少托运行李的重量

**减轻行李技巧**：

- 口袋里只带必需品
- 外套口袋可以装些小东西
- 减少随身包里的冗余物品

## 2. 提前到达与衔接

提前量是应对意外的最佳保险。

### 2.1 到达时间建议

| 交通类型 | 建议提前到达 |
|---------|--------------|
| 国际航班 | 3小时（值机、安检、海关、找登机口） |
| 国内航班 | 2小时（值机、安检、登机） |
| 长途火车 | 1小时（取票、安检、找站台） |
| 短途火车 | 30分钟（提前到站即可） |

**特殊情况加时**：

- 首次去该机场：+30分钟找路
- 节假日/黄金周：+30分钟排队
- 有特殊物品（超过100ml液体、无人机）：+20分钟

### 2.2 路线规划

**出发前确认**：

- 查好从家到机场/火车站的路线
- 预估耗时（预留堵车时间）
- 确认停车位置（如需自驾）

**交通方式选择**：

| 方式 | 适合场景 | 注意事项 |
|------|----------|----------|
| 打车/滴滴 | 行李多、时间紧 | 提前叫车，预留等待时间 |
| 机场大巴 | 费用低、停靠多 | 需要到站点，提前出发 |
| 地铁 | 准时、不堵车 | 确认机场是否通地铁，拖着行李不方便 |
| 朋友送行 | 有人帮忙拿行李 | 提前约好时间 |

### 2.3 同行人集合

**多人出行**：

- 约定好集合时间和地点
- 到达机场后第一时间报平安
- 明确登机口的集合时间

**家人送行**：

- 告别时检查好随身物品
- 不要遗留物品在车里

## 3. 途中安全习惯

旅途中保持警惕，是安全的第一道防线。

### 3.1 财物安全

**分散原则**：

- 现金分开放：身上一部分、证件包里一部分、行李里一部分
- 信用卡分开放：钱包一张、证件包一张、行李里一张

**随身包安全**：

- 人多的地方把包放在前面
- 斜挎包/腰包比双肩包更安全
- 手机不要放在后口袋

**警惕情况**：

- 主动搭讪的陌生人
- "好心人"帮忙拿行李
- 假装问路实则分散注意力
- 拥挤推搡中趁机下手

### 3.2 信息安全

**不要透露**：

- 完整行程（不要说"我一个人在XX旅行"）
- 具体住宿地址（说个大概区域即可）
- 同行人信息（尤其是女性单独旅行）

**社交媒体**：

- 不要实时发位置（到家再发）
- 照片可以延迟发（设为"仅好友可见"）
- 不要公开完整的酒店地址

**遇到危险**：

- 保持冷静，判断情况
- 寻求帮助：警察、景区工作人员、店员
- 拨打当地报警电话
- 联系使领馆

### 3.3 旅途中的日常安全

**餐厅和咖啡馆**：

- 不要用手机占座（容易被顺手牵羊）
- 包和行李在视线范围内
- 离开时检查随身物品

**公共交通**：

- 座椅上方行李架不是最安全的（可能被翻）
- 重要物品放脚边或胸前
- 下车前提前准备，不要慌

**住宿**：

- 进入房间后检查门锁、窗户
- 不轻易开门（确认身份再开）
- 贵重物品放酒店保险箱

## 4. 应急联络网

出发前准备好应急联络清单，关键时刻能救命。

### 4.1 紧急电话清单

**必须记录的电话**：

| 类型 | 电话号码 | 记录位置 |
|------|----------|----------|
| 当地报警 | 各国不同（如日本110、中国香港999） | 手机快速拨号 |
| 急救电话 | 各国不同（120/999/112） | 手机快速拨号 |
| 使馆领事保护 | 查目的地使领馆 | 手机+纸质 |
| 保险公司救援 | 保单上24小时热线 | 手机+纸质 |
| 家人 | 国内紧急联系人 | 手机通讯录 |

**各国报警电话参考**：

```
日本：110（报警）、119（急救/火灾）
韩国：112（报警）、119（急救/火灾）
泰国：191（报警）、1669（急救）
美国：911
欧洲：112（通用紧急电话）
英国：999
法国：17（报警）、15（急救）
新加坡：999
马来西亚：999
越南：113
```

### 4.2 使领馆信息

**提前记录**：

- 使馆名称（中英文）
- 24小时领事保护热线
- 地址（英文和当地语言）
- 官网和社交媒体账号

**使领馆能帮什么**：

- 护照遗失补办旅行证
- 遇到人身安全威胁时提供帮助
- 紧急联系家人
- 提供当地律师、医生信息
- 在重大自然灾害时协助撤离

### 4.3 保险信息

**记录内容**：

- 保险公司名称
- 保单号
- 24小时救援电话
- 报案方式（App/电话/网站）

**需要时**：

- 第一时间联系保险公司
- 说明情况，请求指引
- 保留所有发票和证据
- 回来后提交理赔

### 4.4 银行卡挂失

**提前记录**：

```
银行A：[挂失电话]
银行B：[挂失电话]
...
```

**挂失后**：

- 备用卡可在1-3个工作日寄到
- 可以寄到当地酒店前台
- 现金+其他卡暂时撑过这几天

## 5. 抵达后即时动作

落地后的第一件事，决定了你能否顺利开始旅程。

### 5.1 报平安

**第一步操作**：

1. 开机，连上机场WiFi
2. 给家人/紧急联系人发消息："我已安全到达"
3. 分享位置（可选）
4. 告知后续行程安排

**没有WiFi时**：

- 可以用漫游信号发条简短短信
- 或到机场找到公共电话拨打

### 5.2 通讯准备

**电话卡/网络**：

- 机场购买：排队可能长，但立即可用
- 提前买好：到达即插即用
- eSIM：设置好激活

**重要提醒**：

- 确认手机支持当地网络频段
- 购买流量包时确认够用
- 把手机热点分享给同行人（如需要）

### 5.3 前往住宿

**导航准备**：

- 打开离线地图或Google Maps
- 确认酒店地址和交通方式
- 打印或保存酒店名称（给司机看）

**交通选择**：

| 方式 | 特点 | 适合场景 |
|------|------|----------|
| 机场快线/地铁 | 便宜、准时 | 时间充裕、行李不多 |
| 出租车 | 方便、舒适 | 行李多、晚上到达 |
| 接送机服务 | 预付费、不用等 | 提前预订好的 |
| 打车软件 | 价格透明 | 有当地账号或可付现金 |

**天黑前入住**：

- 尽量在天黑前到达酒店
- 避免夜间拖着行李在陌生街道找路
- 如果到达较晚，选择机场直接打车

## 6. 心理调整与心态建设

旅行中保持好心态，比任何攻略都重要。

### 6.1 接受意外

**常见意外**：

- 航班延误/取消
- 找不到路
- 餐厅关门
- 天气不好
- 买到假票

**正确态度**：

- 意外是旅行的一部分
- 把它当故事，而不是事故
- 保持灵活，快速调整

### 6.2 旅行的目的

**重新审视**：

- 不是"打卡"，是"体验"
- 没有去某个景点不会怎样
- 迷路也是一种探索

**降低预期**：

- 不要被滤镜骗了对目的地的期待
- 真实的旅行比照片更丰富
- 接受"不好玩"的地方

### 6.3 留白与放松

**不要赶**：

- 每天留出空白时间
- 享受"无所事事"
- 允许自己睡个懒觉

**接受变化**：

- 行程可以随时调整
- 删掉某些景点也没关系
- 留点遗憾是下次再来的理由

### 6.4 独自旅行的心理建设

**安全感**：

- 相信自己的判断
- 不轻易相信陌生人
- 遇到问题不要慌

**独立性**：

- 一个人吃饭不尴尬
- 迷路不害怕
- 不需要迁就别人

**社交**：

- 可以主动搭讪其他旅行者
- 参加当地活动结识朋友
- 也要享受独处的时光

## 7. 常见突发情况应对

### 7.1 航班延误/取消

**处理步骤**：

1. 联系航空公司，了解原因和选项
2. 改签：同一航司可能有下一班，跨航司需要协调
3. 退款：如果决定不飞，申请全额退款
4. 保险理赔：保留相关证明，回来索赔

**需要保留的凭证**：

- 延误/取消证明（机场开具或航空公司盖章）
- 所有相关费用发票（餐饮、住宿、交通）
- 登机牌（原本的和新的）

### 7.2 护照丢失

**处理步骤**：

1. 报警：到当地警察局报案，取得报警证明
2. 联系使领馆：报告丢失，申请旅行证
3. 准备材料：报警证明、照片、身份证明
4. 补发旅行证：可能需要1-3个工作日
5. 调整行程：可能需要改变后续计划

**预防措施**：

- 复印件和电子版存好
- 护照和复印件分开放

### 7.3 行李丢失

**处理步骤**：

1. 在到达厅行李查询处登记
2. 提供行李牌和联系方式
3. 等待查找（通常24-48小时）
4. 如确认丢失，申请赔偿

**防止丢失**：

- 行李内放联系方式标签
- 托运前拍照
- 使用独特的行李牌

### 7.4 生病/受伤

**处理步骤**：

1. 轻微：去当地药房买药
2. 中等：去当地医院门诊
3. 严重：拨打急救电话，联系保险公司

**需要保留**：

- 所有诊断证明
- 医疗费用发票
- 保险理赔材料

### 7.5 被骗/被抢

**处理步骤**：

1. 保证人身安全
2. 报警，取得报警证明
3. 联系使领馆
4. 联系银行冻结账户
5. 保险理赔

**预防措施**：

- 现金分散放
- 重要物品不外露
- 保持警惕
> **小贴士**：把这份清单存手机里随时可查。出发前把"手机锁屏紧急联系人"设置好，万一有情况救援人员可以快速联系家人。遇到任何问题，记住：安全第一，其他都是小事。
