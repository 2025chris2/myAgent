# 安全与健康
## 1. 安全评估与预防

### 1.1 行前安全调查

**官方渠道**：

- 中国外交部领事保护中心：了解目的地安全提醒
- 国家移民管理局：最新出入境政策
- Lonely Planet/Tripadvisor：查看旅行警告

**实时安全信息**：

| 渠道 | 内容 |
|------|------|
| 外交部官网 | 级别安全提醒（1-4级） |
| Twitter/X | 实时事件报道 |
| 本地新闻 | 突发新闻 |
| Reddit travel | 旅行者的实时反馈 |

**避开高风险期**：

- 政治敏感期（大选、重要节日）
- 自然灾害季（台风、雨季）
- 罢工/示威高发期

### 1.2 独行女生安全指南

**住店铁则**：

1. 入住前检查门锁、窗户
2. 用椅子抵门
3. 不挂"女性独自入住"的牌子
4. 房间号不要告诉陌生人
5. 夜晚不单独在走廊

**行走夜路**：

| 场景 | 建议 |
|------|------|
| 走路 | 走人多的主路，避开偏僻小巷 |
| 打车 | 用正规软件，叫车后发给朋友 |
| 背包 | 前背，随时可看到 |
| 手机 | 不边走边看，低头看路 |

**降低被盯上的细节**：

- 穿着低调，不要太暴露
- 不佩戴显眼的首饰
- 不主动和陌生人搭讪
- 遇到跟踪：快步走向人多的地方

### 1.3 财务安全

**防盗策略**：

| 方式 | 适用场景 |
|------|----------|
| 分散放钱 | 所有场景 |
| 防盗腰包 | 人多的地方 |
| 隐藏口袋 | 存放备用金 |
| 酒店保险箱 | 不用的大量现金和证件 |

**信用卡安全**：

- 开通消费提醒
- 不让卡离开视线
- 境外交易限额
- 备用卡放不同地方

## 2. 证件丢失应对

### 2.1 护照丢失流程

**发现护照丢失后**：

```
Step 1: 冷静，先确认是否真的丢失（房间、行李反复找）
Step 2: 如确认丢失，立刻报警
Step 3: 报警后取得报警证明
Step 4: 联系中国使领馆
Step 5: 申请旅行证或补发护照
Step 6: 调整行程（可能无法继续原定行程）
```

**报警要点**：

- 到当地警察局报案
- 取得报警证明（原件+复印件）
- 证明上需注明护照号和丢失地点
- 保留报警证明，后续理赔可能用到

**补办旅行证**：

| 步骤 | 内容 |
|------|------|
| 准备材料 | 报警证明、照片、身份证明、护照复印件（如有） |
| 前往使领馆 | 填写申请表 |
| 等待制证 | 通常1-3个工作日 |
| 费用 | 约$35-50 |
| 有效期 | 一次入境有效 |

### 2.2 钱包被偷处理

**被偷后立即操作**：

```
1. 报警（取得证明）
2. 冻结信用卡（联系银行挂失）
3. 联系使领馆（报备身份）
4. 用备用金/备用卡
5. 必要时联系家人汇款
```

**备用金方案**：

- 提前告诉家人备用金位置
- Western Union汇款（全球网点）
- 支付宝国际汇款（手续费低）
- 西联汇款（到账快）

## 3. 保险选择

### 3.1 能救命 vs 管延误

**必须有的保障**：

| 保障类型 | 说明 | 重要性 |
|----------|------|--------|
| 医疗运送 | 把你从当地送回国 | ⭐⭐⭐⭐⭐ |
| 紧急救援 | 直升机/救护车费用 | ⭐⭐⭐⭐⭐ |
| 医疗费用 | 境外就医费用 | ⭐⭐⭐⭐⭐ |
| 意外伤害 | 意外身故/伤残 | ⭐⭐⭐⭐ |

**锦上添花的保障**：

| 保障类型 | 说明 | 重要性 |
|----------|------|--------|
| 航班延误 | 延误X小时赔付 | ⭐⭐⭐ |
| 航班取消 | 赔付已支付费用 | ⭐⭐⭐ |
| 行李丢失 | 赔付生活用品 | ⭐⭐ |
| 旅行变更 | 取消/缩短行程 | ⭐⭐ |

**高风险活动免责**：

- 深潜、滑雪、跳伞等高风险活动
- 部分保险不覆盖，需要额外购买
- 买之前仔细看免责条款

### 3.2 保险挑选要点

**看什么**：

1. 医疗保额：建议不低于30万（发达国家50万+）
2. 紧急救援：是否有24小时中文热线
3. 医疗运送：是否包含直升机和送返
4. 高风险活动：是否覆盖你的活动
5. 理赔流程：是否便捷（App报案？）

**推荐保险公司**：

- 安联（Allianz）：救援服务好
- 美亚（AIG）：产品全面
- 史带（Starr）：境外险专业
- 安盛（AXA）：全球救援网络

## 4. 健康问题应对

### 4.1 飞行健康

**预防血栓**：

| 方法 | 说明 |
|------|------|
| 每2小时起身活动 | 走廊走动 |
| 做脚踝运动 | 转动脚踝，促进血液循环 |
| 多喝水 | 避免脱水 |
| 避免酒精咖啡因 | 可能加重脱水 |
| 穿弹力袜 | 长途飞行推荐 |

**防晕机**：

- 起飞前30分钟吃晕机药
- 选择靠窗座位（颠簸感小）
- 看向窗外远处地平线
- 避免读书看手机

### 4.2 高原反应

**适用地区**：

- 西藏、青海、四川藏区
- 南美（安第斯山脉）
- 非洲乞力马扎罗
- 尼泊尔ABC/EBC

**预防用药**：

| 药物 | 用法 | 说明 |
|------|------|------|
| 红景天 | 出发前5天开始吃 | 预防为主 |
| 高原安 | 按说明书 | 到达后服用 |
| 乙酰唑胺 | 250mg/次，2次/天 | 需医生处方 |

**正确应对**：

1. 缓慢上升：海拔不超过3000米/天
2. 多喝水：高原干燥
3. 不剧烈运动
4. 保证睡眠
5. 如有严重症状（下撤是唯一选择）

### 4.3 晕车晕船

**无瞌睡感的选择**：

| 药物 | 成分 | 优点 |
|------|------|------|
| 茶苯海明（乘晕宁） | 第一代抗组胺 | 效果强但可能嗜睡 |
| 甲磺酸倍他司汀 | 第二代抗组胺 | 嗜睡感低 |
| 生姜片/姜糖 | 天然 | 无副作用 |

**非药物方法**：

- 坐在船中部或车前部
- 看远处的地平线
- 开窗通风
- 薄荷味的东西

### 4.4 动物咬伤处理

**巴厘岛猴子抓伤**：

| 步骤 | 具体做法 |
|------|----------|
| 立即清洗 | 用肥皂水冲洗15分钟 |
| 消毒 | 碘伏或酒精 |
| 就医 | 联系保险，寻求医疗帮助 |
| 狂犬疫苗 | 通常需要5剂，需在当地完成 |
| 记录 | 保留所有就诊记录和发票 |

**其他动物**：

- 狗：同样处理，可能需要狂犬疫苗
- 蛇：不要用嘴吸毒，立即就医
- 水母：醋冲洗，不要用淡水

### 4.5 热带国家饮食安全

**街边摊选择技巧**：

| 观察点 | 放心 | 不放心 |
|--------|------|--------|
| 排队的人 | ✅ 本地人在排队 | ❌ 没有本地人 |
| 食材 | ✅ 生的当场做 | ❌ 做好很久了 |
| 现场 | ✅ 整洁、苍蝇少 | ❌ 脏乱差 |
| 水源 | ✅ 用处理过的水 | ❌ 冰块可能不干净 |

**安全饮食**：

- 剥皮的水果比切块安全
- 热的比冷的安全（微生物被杀）
- 瓶装水刷牙
- 不喝自来水和街头冰块

## 5. 突发情况应对

### 5.1 台风、地震应对

**台风**：

| 阶段 | 行动 |
|------|------|
| 预警 | 储备食物和水，留在室内 |
| 登陆 | 远离窗户，准备手电筒 |
| 经过时 | 留在建筑内部最安全的位置 |
| 过后 | 确认安全后再外出 |

**地震**：

```
国际通用"伏地、遮挡、手抓牢"：
1. 伏地：趴下
2. 遮挡：躲在桌子等坚固物体下
3. 手抓牢：抓紧遮挡物
```

**海啸**：

- 如果在海边感觉到地震，立刻往高处跑
- 不要等官方预警
- 往内陆跑，越远越好

### 5.2 游行示威应对

**避险原则**：

1. 不围观、不参与
2. 迅速离开现场
3. 避开主要道路和广场
4. 找安全建筑躲避
5. 保持通讯畅通

**应对措施**：

- 提前了解当地政治情况
- 关注当地新闻和社交媒体
- 提前储存食物和水
- 保持低调，不穿显眼政治标识

### 5.3 报警与国际通用电话

**全球通用**：

- **112**：GSM标准，全球通用紧急号码，自动转接到当地急救

| 国家 | 报警 | 急救 | 火警 |
|------|------|------|------|
| 中国香港 | 999 | 999 | 999 |
| 日本 | 110 | 119 | 119 |
| 泰国 | 191 | 1669 | 199 |
| 新加坡 | 999 | 995 | 995 |
| 美国 | 911 | 911 | 911 |
| 英国 | 999 | 999 | 999 |
| 欧洲 | 112 | 112 | 112 |

**不会外语怎么报警**：

- 学习目的地语言的"救命"发音
- 用手机翻译App说"Help me call the police"
- 指给路人看112或当地报警电话
- 发短信给112（部分国家支持）

### 5.4 夜间交通

**深夜航班落地**：

| 选项 | 优点 | 缺点 |
|------|------|------|
| 机场过夜等到天亮 | 安全 | 累 |
| 预约接机服务 | 预订好直接走 | 需要提前订 |
| 机场出租车 | 随时有 | 价格可能高 |
| 机场快线+地铁 | 便宜 | 需要研究路线 |

**偏远住宿怎么办**：

- 白天提前去踩点
- 预约正规出租车
- 用Uber/Bolt等软件
- 多人拼车

## 6. 饮品安全

### 6.1 防下药

**夜店酒吧安全**：

| 技巧 | 说明 |
|------|------|
| 酒杯不离手 | 去厕所回来换新酒 |
| 不要接受陌生人给的酒 | 哪怕是打开的 |
| 让朋友陪同 | 互相照应 |
| 点自己的酒 | 看着调酒师调 |
| 有问题立刻离开 | 相信直觉 |

**如何判断被下药**：

- 突然很晕
- 意识模糊
- 失去判断力
- 醒来后不记得发生了什么

**万一发生**：

- 立刻联系朋友或报警
- 去医院检查
- 保留证据（杯子、酒）
- 联系使领馆

### 6.2 交通规则

**各地区交通习惯**：

| 地区 | 靠哪边走 | 备注 |
|------|----------|------|
| 中国 | 靠右 | 右侧通行 |
| 英国 | 靠左 | 左舵车 |
| 日本 | 靠左 | 右舵车 |
| 澳大利亚 | 靠左 | 右舵车 |
| 美国 | 靠右 | 右舵车 |
| 欧洲 | 靠右 | 右舵车 |

**租车注意**：

- 确认驾照+翻译件可用
- 靠左行驶国家（英国、日本、澳大利亚）需要适应期
- 靠右行驶国家更接近国内习惯

### 6.3 租摩托车

**是否需要驾照**：

| 国家/地区 | 要求 |
|-----------|------|
| 泰国 | 需要国际驾照或泰国驾照 |
| 越南 | 中国驾照不被认可，风险自担 |
| 巴厘岛 | 需要国际驾照 |
| 澳洲 | 需要国际驾照 |

**头盔**：

- 几乎所有国家都强制要求
- 没有头盔被罚款
- 建议自带，质量更好

## 7. 手机相关安全

### 7.1 手机被抢

**立即做三件事**：

```
1. 冻结账号：致电银行冻结相关账号
2. 挂失手机：联系运营商挂失SIM卡
3. 远程锁定/清除：iCloud Find My iPhone 或 Google Find My Device
```

**预防措施**：

- 开启手机定位功能
- 记录手机序列号（IMEI）
- 不在街上用手机暴露贵重物品

### 7.2 迷路求助

**迷路时怎么办**：

1. 打开离线地图（Google Maps提前下载）
2. 找附近的商铺/警察局问路
3. 用翻译App说"我在哪里？我怎么去[目的地]？"
4. 打电话给酒店（提前存好地址和电话）

**防止迷路**：

- 出发前把酒店地址写在纸上
- 开启手机定位
- 每隔一段截一张地图
- 设置好紧急联系人快捷键

## 8. 紧急联络卡

### 8.1 紧急联络卡内容

**必须包含**：

```
姓名：[中文名+英文名]
血型：[填写]
紧急联系人：[关系] [电话]
我的保险：[公司名] [保单号] [救援电话]
我的过敏：[如有]
我的病史：[如有]
中国使领馆：[电话]
```

**如何携带**：

- 放在钱包内层
- 手机壳后面
- 设为手机锁屏壁纸
- 告诉同行人

### 8.2 大使馆电话设置

**一键拨打**：

- iPhone：添加到通讯录，点击即可拨打
- Android：创建桌面快捷方式
- 或设置Siri快捷命令
