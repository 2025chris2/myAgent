# 预订落实类
## 1. 锁定大交通

交通是旅行的骨架，一旦确定就不要轻易变动（除非发现了更好的选择）。

### 1.1 机票预订要点

**预订前确认**：

- 出发/到达城市和日期正确
- 航班时间是否适合（不要太早或太晚）
- 转机时间是否足够（国际转机至少3小时）
- 航站楼信息（同一城市不同航站楼可能距离很远）

**廉航特别注意**：

- 行李额：是否含托运？手提行李尺寸和重量限制？
- 座位选择：是否需要额外付费选座？
- 餐食：是否不含餐？机上餐食价格？
- 改期政策：是否能改期？改期费多少？

**付款后检查**：

- 邮箱是否收到行程单（ITINERARY）
- 订单号是否正确
- 乘客姓名拼写是否正确（和护照一致）
- 航班时间、日期、航班号是否正确

### 1.2 火车票预订

**国内火车票**：

- 提前15天开始放票，热门线路尽早抢
- 12306官网/App或携程/去哪儿代购
- 注意学生票优惠（需要磁条）
- 高铁动车的静音车厢和普通车厢区别

**国际火车票**：

- 欧洲铁路：提前3个月买早鸟票最便宜；欧铁通票适合多国游
- 日本：新干线提前1个月买；JR Pass适合全国跑
- 东南亚：现场买可能比提前买便宜，但旺季建议提前

**预订后确认**：

- 打印电子票或保存电子票PDF
- 确认站台信息（有时会临时变更）
- 了解是否需要取票（部分国家凭二维码即可进站）

### 1.3 船票预订

- 海岛跳岛：旺季船票紧张，提前1-2周预订
- 跨国渡轮：欧洲（英国-法国）、东南亚（泰国-马来西亚）
- 内河游轮：长江三峡、莱茵河、多瑙河

### 1.4 行李额与政策

**托运行李**：

- 经济舱通常23kg一件，部分航司30kg
- 超重费：超1kg可能收取几十到几百元
- 超件费：第二件托运很贵

**手提行李**：

- 一般7-10kg，尺寸限制（55x40x20cm左右）
- 廉价航空要求更严格，超重或超尺寸会被拒

**退改政策**：

- 灵活票：可免费改期，差价多退少补
- 标准票：改期收费，退票有限制
- 特价票：不可改不可退

## 2. 预订住宿

住宿直接影响旅行质量和休息质量，需要认真挑选。

### 2.1 位置确认

**用地图工具检查**：

- 从住宿到最近地铁站/公交站步行距离（不超过500米为佳）
- 到主要景点的距离和交通方式
- 周边便利店、餐厅、药房配置
- 夜间是否有路灯、是否安全

**地图测量方法**：

- Google Maps：右键点击住宿→选择"测量距离"→点击地铁站→看距离
- 苹果地图：查看路线步行时间

### 2.2 筛选条件

| 场景 | 必备条件 | 加分条件 |
|------|----------|----------|
| 带老人 | 有电梯、无台阶、靠近地铁 | 无障碍房型 |
| 自驾 | 提供免费停车 | 停车场安全 |
| 亲子 | 允许儿童入住、可加床 | 儿童泳池、亲子活动 |
| 情侣 | 位置好、景观佳、隔音好 | 蜜月布置、浪漫晚餐 |
| 独自旅行 | 位置安全、评价好、工作人员靠谱 | 公共区域可社交 |

### 2.3 预订平台对比

| 平台 | 优点 | 缺点 |
|------|------|------|
| Booking | 房源多、免费取消多、有Genius会员优惠 | 价格有时不是最优 |
| Agoda | 亚太地区房源多、常有折扣 | 取消政策较复杂 |
| Airbnb | 独特民宿、有厨房、适合多人 | 卫生参差、可能不符预期 |
| 携程/去哪儿 | 中文界面、客服方便、价格有时最低 | 房源质量参差 |
| 官网预订 | 最低价保证、有会员优惠 | 价格不如OTA优惠 |

**价格对比技巧**：

- 同一酒店在多个平台比较价格
- 官网价格+会员折扣可能更低
- 关注各平台的大促（618、双11、周三会员日）

### 2.4 预订注意事项

**预订时确认**：

- 入住/退房时间是否匹配行程
- 取消政策（免费取消截止日期）
- 住宿规定（能否吸烟、能否带宠物）
- 额外费用（城市税、清洁费、服务费）

**预订后操作**：

- 截图保存预订确认页
- 记录订单号、酒店地址、电话
- 查看是否收到确认邮件
- 抵达前1周再次确认（部分酒店需要）

### 2.5 特殊住宿预订

**青旅床位**：

- 预订时选好房型（混合/女生/男生）
- 查看是否有储物柜、是否需要自带锁
- 注意退房时间和熄灯时间

**寺庙住宿/禅修**：

- 部分需要提前几个月申请
- 斋饭、早课等安排是否包含

**度假村**：

- 一价全包vs非全包要算清楚
- 提前订活动（浮潜、皮划艇等）可能优惠

## 3. 提前预约热门项目

有些体验必须提前预约，否则到了现场可能根本没有名额。

### 3.1 必须提前预约的项目

**限流景点**：

| 景点 | 提前预约时间 | 预约方式 |
|------|--------------|----------|
| 故宫 | 提前10天 | 故宫博物院官网/微信 |
| 卢浮宫 | 提前1个月 | 官网预约时间段 |
| 高迪建筑（圣家堂等） | 提前1-3个月 | 官网预约 |
| 日本环球影城 | 提前购买门票+快速票 | 官网/飞猪 |
| 新西兰萤火虫洞 | 旺季需提前2周 | 官网预约 |

**特殊体验**：

| 类型 | 说明 | 建议提前 |
|------|------|----------|
| 米其林餐厅 | 热门餐厅提前1-3个月预约 | 1-3个月 |
| 和服换装 | 京都等地旺季提前2-3周 | 2-4周 |
| 潜水课程 | 需要健康证明和培训 | 2-4周 |
| 热气球 | 天气依赖，可能需要备选日期 | 1-2周 |
| 滑雪课程 | 教练资源有限，旺季需抢 | 1-2周 |

### 3.2 一日游/多日团选择

**选择标准**：

- 正规旅行社运营（有资质、口碑好）
- 包含内容清晰（门票、包餐否）
- 集合时间和地点明确
- 团费包含什么、不包含什么清楚

**预订渠道**：

- 出发前线上预订：飞猪、携程、Klook客路
- 当地酒店前台预订：可能有更多选择
- 不建议临时在景点门口订，可能被宰

**确认事项**：

- 集合时间地点（到酒店接还是自己去）
- 导游语言（是否会说中文）
- 是否需要额外付小费
- 取消和改期政策

### 3.3 预订凭证管理

**所有预订完成后**：

- 截图保存（包含订单号、日期、价格）
- 整理到一个文件夹（按日期或类别）
- 发送一份到自己的邮箱备份
- 离线保存到手机本地（PDF或截图）

**凭证清单**：

- [ ] 国际机票行程单
- [ ] 火车票/船票
- [ ] 酒店预订确认单
- [ ] 景点门票/预约确认
- [ ] 一日游/体验预约确认
- [ ] 租车确认单
- [ ] 旅行保险单

## 4. 办理必要证件

证件办理需要时间，务必提前开始。

### 4.1 护照相关

**护照换发**：

- 有效期不足6个月需更换
- 空白页不足2页需更换
- 护照损坏需更换
- 办理周期：一般10个工作日，可加急

**港澳通行证/台湾通行证**：

- 确认签注是否有效（一年一次/两次/多次）
- 确认签注类型（个人游/团队游）
- 有效期内签注用完需重新办理

### 4.2 签证申请

**材料准备清单**：

| 材料类型 | 说明 | 注意事项 |
|----------|------|----------|
| 护照 | 有效期6个月以上 | 原件+复印件 |
| 照片 | 符合规格（白底、免冠） | 各国要求不同 |
| 签证申请表 | 官网下载或在线填写 | 用英文/当地语言填写 |
| 在职证明 | 公司抬头纸、盖公章 | 在有效期内的 |
| 银行流水 | 3-6个月流水 | 余额需达到要求 |
| 行程单 | 机酒预订单、每日行程 | 真实可查 |
| 保险单 | 境外医疗险 | 部分国家要求 |
| 照片 | 符合规格 | 不同国家尺寸不同 |

**递签后跟踪**：

- 记录递签日期和预计出签日期
- 保持电话畅通（可能补材料）
- 出签后仔细核对信息（姓名、日期、次数）

### 4.3 国际驾照认证件

**需要的情况**：

- 自驾目的地要求IDP（International Driving Permit）
- 部分国家认可中国驾照+翻译件

**办理方式**：

- 中国驾照翻译公证件：公证处办理，3-5工作日
- 国际驾照认证件：租租车/惠租车平台免费办理
- 淘宝也有代办服务

**注意事项**：

- 部分国家（如美国部分州）不认可中国驾照
- 日本需要IDP，中国大陆不颁发IDP

## 5. 购买旅行保险

旅行保险是应对意外的最后防线，看似多余实则必要。

### 5.1 必须包含的保障

**医疗保障**：

- 保额建议不低于30万元（发达国家建议50万+）
- 覆盖门诊和住院
- 包含意外伤害医疗

**紧急救援**：

- 医疗运送和送返（最重要的一项）
- 遗体运送
- 24小时全球紧急救援热线

**行程阻碍**：

| 保障内容 | 说明 |
|----------|------|
| 航班延误 | 延误超过X小时赔付 |
| 航班取消 | 赔付已支付未使用的费用 |
| 行李延误/丢失 | 赔付必要生活用品购置费 |
| 旅行变更 | 因突发情况取消/缩短行程 |

**财物与责任**：

- 随身财产被盗抢
- 银行卡盗刷
- 意外导致第三方人身/财产损失

### 5.2 选择保险的要点

**看保险公司**：

- 选择有境外救援服务经验的公司
- 安联、美亚、史带、安盛等都有不错的旅行险
- 查看救援服务评价

**看条款细节**：

- 免赔额多少
- 既往病史是否除外
- 高风险活动是否覆盖（如潜水、滑雪）
- 理赔流程是否清晰

**按目的地选择**：

- 申根国家：必须购买覆盖申根的保险
- 美国/加拿大：医疗费用高，建议高保额
- 东南亚：医疗险保额30万足够
- 海岛游：如果有高风险活动选含此类活动的险

### 5.3 购买后的操作

**保存凭证**：

- 保险单PDF（打印一份随身带）
- 保险条款PDF
- 24小时救援热线电话（手机存好）

**告知同行人**：

- 把保险信息和紧急联络方式分享给同行人
- 或分享给家人备份
> **小贴士**：预订落实阶段建议用Excel或项目管理工具跟踪进度，列出每个预订的状态（待确认/已确认/已完成）。所有重要凭证建议打印一份随身带、存手机一份、发邮箱一份、云盘一份，做到万无一失。
