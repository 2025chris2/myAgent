# 资金与支付类
## 1. 现金准备

现金在很多场景仍然不可替代，但现代旅行不需要携带大量现金。

### 1.1 兑换当地货币

**出发前兑换**：

- 在国内银行提前预约兑换
- 不需要换太多，留够前两天使用即可
- 机场汇率通常不如市区好，但紧急时也可在机场换

**兑换金额估算**：

| 目的地类型 | 建议兑换金额 | 说明 |
|-----------|-------------|------|
| 发达国家（日本、欧美） | 100-200元等值 | 主要是小费、交通、应急 |
| 东南亚（泰国、越南等） | 300-500元等值 | 很多小摊、小店只收现金 |
| 刷卡发达地区 | 可少换 | 几乎所有地方都能刷卡 |

**小面额准备**：

- 尽量要些小面额纸币和硬币
- 用于：购买车票、上厕所、给小费、小摊贩
- 不要全是整钱，小摊贩可能找不开

### 1.2 当地取现

**ATM取现**：

- 到达后可在正规银行ATM取现
- 每次取现有限额，注意手续费
- 银联卡在很多国家都可以用

**手续费说明**：

| 费用类型 | 说明 | 节省方法 |
|----------|------|----------|
| 发卡行手续费 | 境外取款每笔固定费用 | 选择免境外取现手续费的卡 |
| 当地银行费 | 当地ATM所属银行收取 | 找标识"银联"或免除附加费的ATM |
| 货币转换费 | 汇率转换损失 | 使用当地货币直接结算 |

**取现技巧**：

- 一次取够，减少次数就是减少手续费
- 避免在机场ATM取现（手续费通常较高）
- 记好自己卡的每日取现限额

### 1.3 货币知识

**常见货币代码**：

- CNY 人民币、USD 美元、EUR 欧元
- JPY 日元（注意：没有小数点，1日元≈0.05人民币）
- THB 泰铢、VND 越南盾（注意：数值大但实际便宜）
- KRW 韩元、GBP 英镑

**汇率查询渠道**：

- 银行App实时汇率
- XE Currency、Google汇率
- 当地ATM显示的汇率（通常比实时汇率略低）

**换钱避坑**：

- 不在黑市换钱，有假币和欺诈风险
- 不换大额货币，汇率波动风险
- 行程结束前花完剩余外币，或换成下程目的地货币

## 2. 银行卡与信用卡

银行卡是旅行中的主要支付工具，需要提前做好准备。

### 2.1 出发前告知银行

**必须做的操作**：

- 在银行App或客服热线报备出行时间和目的地
- 开通境外消费功能
- 设置境外交易限额（或调高）

**报备内容**：

- 出行时间段（具体日期）
- 目的地国家和地区
- 预计消费金额（可估算）

**不报备的后果**：

- 卡片可能被银行风控系统锁定
- 在境外刷卡会被拒
- 解锁需要联系银行（可能国际长途）

### 2.2 信用卡功能确认

**境外支付功能**：

- 确保信用卡已开通境外网上交易
- 部分银行需要主动开通"外币交易"功能
- 小心网上预约购票（如日本景点门票）可能需要验证

**借记卡境外取现**：

- 借记卡需开通境外取现功能
- 部分银行借记卡境外取现有每日限额
- 借记卡建议作为备用，主要用信用卡

### 2.3 手续费了解

**刷卡手续费**：

- 货币转换费（1.5%-2%不等）
- 部分银行境外刷卡有额外手续费
- 选择外币直接结算（避免双重转换）

**取现手续费**：

| 卡片类型 | 取现手续费 | 说明 |
|---------|-----------|------|
| 免境外费借记卡 | 0-10元/笔 | 如华夏银行、宁波银行等 |
| 普通信用卡 | 1%-3%+固定费 | 取现额2%-5%不等 |
| 高端信用卡 | 部分免手续费 | 但可能有年费 |

**节省手续费策略**：

- 优先刷卡，少用现金
- 大额消费用信用卡（大额取现手续费高）
- 选择免除手续费的高端卡（适合经常出境者）

### 2.4 卡片分散策略

**携带至少两张卡**：

- 建议至少带两张不同卡组织的卡
- 例如：一张Visa/Mastercard + 一张银联
- 不同卡在不同场景可用（银联在亚洲覆盖广，Visa/Mastercard在欧美接受度高）

**分开存放**：

- 一张常用，一张备用
- 备用卡放在行李不同位置或酒店保险箱
- 不要把两张卡放在同一个地方

## 3. 移动支付与电子钱包

移动支付在很多地区已经非常普及，了解当地支持情况可以让支付更便捷。

### 3.1 支付宝和微信支付

**境外覆盖情况**：

| 支付方式 | 主要覆盖地区 |
|---------|-------------|
| 支付宝 | 日本、韩国、东南亚（泰国、越南、马来西亚） |
| 微信支付 | 日本、韩国、东南亚主要旅游城市 |
| 两者 | 大型商场、连锁店、机场免税店接受度高 |

**使用方式**：

- 提前绑定支持境外使用的银行卡
- 到达后打开付款码，直接扫码支付
- 按实时汇率结算，自动购汇

**注意事项**：

- 小摊贩、小店可能不支持
- 可能有单笔或单日限额
- 查看APP内的汇率，选择最优

### 3.2 Apple Pay / Google Pay

- 在境外接受 contactless 支付的商店可用
- 直接从绑定的信用卡扣款
- 可能比实体卡有额外安全保护

### 3.3 当地电子钱包

**日本**：

- 交通卡（Suica、PASMO）：可充值用于交通和购物
- 需要现金充值，机场和便利店可充值

**东南亚**：

- LINE Pay（泰国）
- GrabPay（东南亚多国）

**欧洲**：

- Apple Pay/Google Pay 在很多地方可用
- 部分国家有本地支付App

## 4. 安全备份

旅行中丢失钱包/卡片是最糟糕的情况之一，提前做好备份能救命。

### 4.1 卡片信息记录

**必须记录的信息**：

- 卡片正面：卡号、有效期、持卡人姓名
- 卡片背面：CVV安全码（但不要和卡片放一起）
- 银行境外服务热线

**记录方式**：

- 手机加密相册（建议新建一个带密码的相册）
- 加密云笔记（如有道云笔记、印象笔记密码保护）
- 纸质备份（放在家里，不随身带）

**记录格式示例**：

```
银行：中国银行
卡片类型：Visa信用卡
卡号：**** **** **** 1234
有效期：XX/XX
持卡人：张三
境外客服：+86-10-95566
```

### 4.2 备用金方案

**应急备用金**：

- 出发前告诉家人你有备用现金在哪里
- 紧急情况下家人可以汇款给你

**汇款渠道**：

- 银行电汇（慢，费用高）
- 西联汇款（较快，到账快）
- 支付宝国际汇款（到账快，费用相对低）

### 4.3 卡片丢失应对

**第一时间**：

1. 联系银行境外客服（提前存好号码）
2. 冻结/挂失卡片
3. 申请补发（可寄到当地酒店）

**备用方案**：

- 备用卡可在1-2个工作日补发
- 现金+其他卡可以撑过这几天
- 提前让银行知道行程，紧急补卡更容易

### 4.4 防盗技巧

**现金分散**：

- 不要把所有现金放在一个地方
- 贴身放一些（应急用）
- 行李里放一些（备用）

**支付时**：

- 在餐厅或商店，让服务员拿走卡，不要让卡离开视线
- 小心"复制卡片"的骗局
- 输入密码时遮挡

**警惕诈骗**：

- 不要轻信陌生人的"帮你刷卡"提议
- 警惕假扮银行客服的诈骗电话
- 警惕"你的账户有问题"类短信

## 5. 预算控制技巧

旅行中超支是常见问题，提前设定预算并执行能避免回来吃土。

### 5.1 每日预算设定

**设定方法**：

- 总预算 ÷ 行程天数 = 每日预算
- 其中包含：餐饮+交通+小额购物+娱乐

**预算分配示例（总预算1万元，7天行程）**：

| 类别 | 金额 | 说明 |
|------|------|------|
| 每日餐饮 | 150-200元 | 中午简餐+晚上正餐 |
| 每日交通 | 50-100元 | 地铁公交+偶尔打车 |
| 小额购物 | 50-100元 | 零食、纪念品 |
| 娱乐体验 | 50-100元 | 门票、体验活动 |
| **每日合计** | **300-500元** | |

### 5.2 记账方法

**旅行记账App推荐**：

- 旅行账本（专注旅行预算）
- 随手记（通用记账）
- 支付宝/微信自带记账（消费记录）

**每日记录**：

- 每笔消费随手记下
- 睡前看一眼今日支出
- 超出预算时及时调整

### 5.3 超支应对

**调整策略**：

- 降低餐饮标准（从餐厅转战快餐/便利店）
- 减少购物支出
- 用免费体验替代付费项目

**心理调整**：

- 旅行体验比物质重要
- 超支买的东西可能回来就后悔
- 留下遗憾是下次再来的理由

## 6. 支付方式汇总表

| 场景 | 推荐支付方式 | 说明 |
|------|-------------|------|
| 大型商场、免税店 | 信用卡/支付宝/微信 | 可能有返现/优惠 |
| 连锁餐厅、便利店 | 信用卡/交通卡/移动支付 | 方便快捷 |
| 街边小店、市场 | 现金 | 部分只收现金 |
| 交通（地铁、公交） | 交通卡/当地卡 | 充值方便，不找零 |
| 门票、景点 | 信用卡/支付宝 | 提前官网买更便宜 |
| 小费 | 现金 | 准备小面额 |
| 酒店（押金、附加费） | 信用卡预授权 | 确保卡可用 |
> **小贴士**：在信用卡背面签名栏旁边写上"境外挂失电话+86-xxxxxxxxx"，万一丢失方便他人帮忙联系。尽量选择有芯片的卡，芯片卡比磁条卡更安全。旅行前把不用的信用卡临时限额降低，万一出问题损失更小。
