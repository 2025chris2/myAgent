# 家庭事务收尾类

> 出门前把"后方"安置好，能极大减少旅途中突然想起"我有没有关窗户？"带来的焦虑。旅行时心无旁骛的前提是家里一切安好。

---

## 1. 动植物与家务

旅行前把家里的"活物"安排妥当，这是最容易让你分心的牵挂。

### 1.1 植物养护

**拜托他人帮忙**：

- 找可信任的朋友/邻居定期上门浇水
- 留下浇水频率和注意事项（喜阳/喜阴/多少水）
- 提供备用钥匙或门禁卡

**DIY解决方案**：

- 缓滴装置：用倒置的矿泉水瓶，瓶盖扎小孔
- 自动浇水器：淘宝有几十元的自动浇水定时器
- 阳台植物：拉上遮阳网，避免暴晒

**特别提醒**：

- 容易干的植物多浇点，喜水的植物套个保鲜膜保湿
- 怕冷的植物移到室内
- 开花植物可能需要特殊照顾

### 1.2 宠物安置

**选项对比**：

| 方式 | 优点 | 缺点 |
|------|------|------|
| 上门喂养 | 宠物不用离开家，压力小 | 需要信任的人 |
| 宠物店寄养 | 专业照顾，社交机会 | 费用高，环境陌生 |
| 朋友家寄养 | 宠物有人陪，减少分离焦虑 | 需要宠物适应新环境 |
| 带回家 | 父母帮忙照顾 | 父母可能有压力 |

**准备事项**：

```
□ 足够的食物（按天数+3天余量）
□ 猫砂/狗尿垫（多备点）
□ 常用药物（驱虫药、消炎药等）
□ 兽医联系方式
□ 宠物习惯说明书（喂食时间、量、注意事项）
□ 备用钥匙和联系方式
□ 最近的宠物医院地址
```

**特别提醒**：

- 交代清楚宠物的性格和特殊习惯
- 留下紧急联络方式（自己手机、兽医、朋友）
- 走前陪宠物玩一会儿，减少分离焦虑

### 1.3 家务收尾

**必须做的**：

- 扔掉所有垃圾（尤其是湿垃圾、厨余垃圾）
- 清理冰箱里易腐食物（或者请朋友帮忙处理）
- 洗碗洗锅，不留脏餐具
- 倒掉垃圾桶，防止异味

**可选项**：

- 脏衣服洗掉（特别是内衣、袜子）
- 床单被套换洗（回来干净清爽）
- 垃圾袋准备好（回来不用买）

## 2. 水电安全与节能

出门前做一次安全检查，确保家里不会出现漏水、短路等问题。

### 2.1 水源安全

**必须检查**：

- 所有水龙头拧紧
- 检查马桶水箱是否漏水
- 洗衣机进水管和出水管接头是否松动
-热水器是否正常，有无渗漏

**特别提醒**：

- 总水阀可以关掉（如果长时间不在）
- 地漏确保畅通，防止积水

### 2.2 电源与燃气

**电源检查**：

- 拔掉非必须电器的插头：电视、充电器、烧水壶、电饭煲等
- 检查插座是否松动、发热
- 关闭不用的开关
- 如有必要，关闭总电源开关

**安全提示**：

- 不要让充电器长时间插着（有发热风险）
- 大功率电器（空调、热水器）可以关掉总开关

**燃气检查**：

- 检查燃气阀门是否关闭
- 燃气灶开关是否关紧
- 闻一下是否有异味（如果有，联系燃气公司）

### 2.3 门窗检查

**出门前最后检查**：

```
□ 所有窗户关好、锁好
□ 阳台门锁好
□ 大门反锁（防盗门多锁几道）
□ 防盗窗检查（如果有）
□ 不要留钥匙在门锁上
```

**户外区域**：

- 收好阳台上的花盆、杂物（防止掉落）
- 收起晾晒的衣物
- 关闭电动窗帘

## 3. 服务与订阅管理

避免旅行期间服务继续扣费或包裹堆积。

### 3.1 定期配送暂停

**需要暂停的服务**：

| 类型 | 处理方式 |
|------|----------|
| 牛奶配送 | 打电话暂停 |
| 报纸配送 | 暂停或转送 |
| 桶装水配送 | 暂停 |
| 快递上门 | 修改地址或暂存快递柜 |

### 3.2 邮件自动回复

**设置内容示例**：

```
您好！

我目前不在办公室，将于[返回日期]恢复正常工作。

如有紧急事务，请联系[紧急联系人姓名]：[电话]

谢谢！

[你的名字]
```

**设置方式**：

- 各大邮箱（QQ邮箱、163邮箱、Gmail）都有自动回复设置
- 或者让家人/朋友帮忙处理重要邮件

### 3.3 外卖与购物地址

**检查外卖App**：

- 关闭外卖App的默认地址
- 避免旅行期间误下单送到家里

**购物平台**：

- 暂停待发货的订单（如可能）
- 修改默认收货地址
- 或者让邻居/物业帮忙代收

## 4. 信息交接

让家人和信任的人知道你的行程，关键时刻能找到你。

### 4.1 给家人的信息包

**必须告知的信息**：

```
□ 完整行程单（出发/返回日期、航班号、酒店）
□ 酒店名称、地址、电话
□ 旅行保险单（纸质或电子）
□ 保险公司24小时救援热线
□ 中国使领馆领事保护电话（目的地的）
□ 国内紧急联系人电话
```

**分享方式**：

- 把以上信息整理成一个文档，发给家人
- 放在共享云盘（家人可随时查看）
- 纸质版留一份在家中

### 4.2 给邻居/物业

**可选告知**：

- 出门时间（如果时间长）
- 紧急联系方式
- 是否需要帮忙关注家里情况（如有人来访）

**物业托管**：

- 如果物业有代收快递服务，可以提前沟通
- 确保邻居知道有异常可以联系你

### 4.3 安全清单

**贴在门上或冰箱上的清单**：

```
[出发前最后检查]

□ 门窗关好锁好
□ 水龙头关紧
□ 燃气阀门关闭
□ 电源插头拔掉
□ 总开关关闭（如果长时间不在）
□ 垃圾已倒
□ 冰箱易腐食物已清理
□ 植物/宠物已安排好
□ 重要文件备份已上传云端

[紧急联系方式]
本人电话：
家人电话：
物业电话：
```

## 5. 出发前24小时最后检查

出发前一天是最后的机会，确保所有事务都已收尾。

### 5.1 确认清单

**行程确认**：

- [ ] 航班状态再次确认（有无延误取消）
- [ ] 机场接送安排确认
- [ ] 护照、签证确认在有效期内
- [ ] 行李已打包完成

**家中确认**：

- [ ] 门窗已锁
- [ ] 水电燃气已检查
- [ ] 垃圾已清理
- [ ] 植物/宠物已托付
- [ ] 服务已暂停

**信息确认**：

- [ ] 行程单已发给家人
- [ ] 紧急联系人已告知
- [ ] 保险单已打印/已发送

### 5.2 最后收尾

**当天可做的事情**：

- 再检查一遍重要物品
- 给家人打个电话确认他们收到了行程信息
- 放松心情，早点休息

**特别提醒**：

- 出发前给植物/宠物留好足够的水和食物（最后检查）
- 设置好邮件自动回复
- 临时要更改的东西可以拜托信任的人帮忙处理

### 5.3 出发当早

**出发前最后检查**：

- 证件包：护照、手机、钱包、钥匙
- 行李：托运行李已贴好标签，随身包已检查
- 家里：最后看一眼门窗
- 心态：放轻松，准备出发！

## 6. 特别提醒

### 6.1 长期旅行的特别安排

**超过一周的旅行**：

- 可以请家政来打扫一次再出发
- 考虑请物业或邻居定期帮忙检查
- 设置网络摄像头（可选）远程查看家里
- 订阅暂停+邮件自动回复更必要

**超过两周的旅行**：

- 可能需要请人来家里浇花
- 考虑把重要文件、贵重物品放到保险箱或朋友家
- 关闭不必要的服务（如电视、网络可以远程关闭）

### 6.2 极端天气前的特别检查

**台风季节出行**：

- 收好阳台物品
- 关好所有窗户
- 准备防雨措施（如果走之前台风要来）

**暴雨季节出行**：

- 检查排水系统
- 把怕潮湿的东西放到高处

### 6.3 与邻居/朋友的沟通

**信任建立**：

- 提前跟邻居/朋友沟通你的需求
- 准备好联系方式和备用钥匙
- 不强求，感恩帮忙
- 回来后可以送个小礼物或请吃饭
> **小贴士**：出发前把家里拍几张照片存档，回来时对照检查有没有异常。可以设置一个"旅行出发"清单文档，出发前按项勾选，避免遗漏。把家里的水电气总开关位置告诉家人，万一有情况可以远程处理。
