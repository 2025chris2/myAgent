# 物品打包类

> 高效打包的核心是按场景分类，而不是按物品类别。用"如果缺少它会很麻烦"作为判断标准。打包的原则是：宁可少带，不要多带；轻便实用，够用就好。

---

## 1. 证件包（绝不能托运）

这是最重要的包，随身携带，寸步不离。

### 1.1 证件类

| 物品 | 说明 |
|------|------|
| 护照及复印件 | 原件放最里层安全位置，复印件备用 |
| 签证/签注 | 有效页，落地签准备好相关材料 |
| 身份证 | 国内段交通备用 |

### 1.2 票据类

| 物品 | 说明 |
|------|------|
| 机票/车票 | 打印一份，电子版手机存一份 |
| 酒店确认单 | 地址和电话清晰可见 |
| 保险单 | 含紧急救援热线 |
| 行程单 | 包括每日安排、预订确认 |

### 1.3 应急物品

| 物品 | 说明 |
|------|------|
| 紧急联系人卡 | 姓名、电话、与本人关系 |
| 2寸证件照 | 2张，补办证件用 |
| 一支笔 | 填入境卡用 |
| 现金 | 少量当地币和人民币 |
| 信用卡 | 1-2张 |

### 1.4 证件包整理技巧

**安全放置**：

- 放在防盗腰包或贴身口袋
- 或放在前背包的最内层
- 每天睡前检查是否在原位

**不要托运**：任何证件都不要放在托运行李里。

## 2. 衣物与鞋子

衣物是行李的主体，按场景和天气准备。

### 2.1 基础穿搭原则

**洋葱式穿衣法**：

```
外层：防风防水外套（应对天气变化）
中层：保暖层（抓绒/卫衣，根据目的地）
内层：速干基础（T恤、衬衫）
```

**数量控制**：

- 出行天数×0.7 = 上衣数量（带可搭配的基础款）
- 内衣裤袜子按天数准备或略少（可洗）
- 外套带1-2件够了

### 2.2 不同目的地配置

**热带海岛（东南亚、海岛）**：

```
□ 短袖T恤 ×3-4
□ 短裤/沙滩裤 ×2-3
□ 泳衣/泳裤 ×1-2
□ 薄外套 ×1（室内空调冷）
□ 防晒衣/防晒袖套
□ 晾衣绳或便携衣架
```

**温带城市（欧洲、日韩、中国城市春秋）**：

```
□ 长袖T恤/衬衫 ×4-5
□ 牛仔裤/休闲裤 ×2-3
□ 薄外套 ×1
□ 轻便外套/风衣 ×1
□ 换洗内衣 ×对应天数
```

**寒冷地区（北欧、冬季日本）**：

```
□ 保暖内衣 ×3-4套
□ 毛衣/厚卫衣 ×2-3
□ 羽绒服（可压缩最好）
□ 保暖裤/毛裤 ×2-3
□ 手套、围巾、帽子
□ 暖宝宝（怕冷的人）
```

### 2.3 鞋子选择

**标配三双**：

| 类型 | 用途 | 选择建议 |
|------|------|----------|
| 主鞋 | 每天暴走 | 好走、防滑、轻便、平底/运动鞋 |
| 休闲鞋 | 约会/晚餐 | 轻便、可搭配的帆布鞋/休闲皮鞋 |
| 拖鞋 | 飞机/酒店/浴室 | 轻便折叠款 |

**特别情况**：

- 海岛：带潜水鞋（保护脚不被珊瑚划伤）
- 雪地：防滑防水的雪地靴
- 雨季：防水鞋或带鞋套

**减少鞋子的技巧**：

- 穿最重/最大的鞋子上飞机
- 主鞋+拖鞋可以应对大部分场景
- 一双好走的主鞋比多双低质鞋有用

### 2.4 配件清单

**常用配件**：

```
□ 墨镜（保护眼睛，还防晒）
□ 帽子（防晒或保暖）
□ 围巾/丝巾（进寺庙保暖，还能当装饰）
□ 折叠衣架 ×2-3（酒店可能没有或有但不够）
□ 晾衣绳（民宿必备）
```

## 3. 常用药品

药品不怕一万就怕万一，带对了能救命。

### 3.1 基础药品清单

| 类别 | 药品名称 | 用途 |
|------|----------|------|
| 感冒药 | 白加黑、感康 | 缓解感冒症状 |
| 退烧止痛 | 布洛芬、对乙酰氨基酚 | 退烧、止痛、缓解姨妈痛 |
| 肠胃药 | 黄连素、蒙脱石散、益生菌 | 腹泻、消化不良、水土不服 |
| 抗过敏 | 氯雷他定、西替利嗪 | 过敏性鼻炎、皮肤过敏 |
| 外伤 | 创可贴、碘伏棉签 | 小伤口处理 |
| 其他 | 润喉片、眼药水 | 喉咙不适、眼睛干涩 |

### 3.2 个人特殊药物

**必须带的药物**：

- 慢性病药物（高血压、糖尿病、哮喘等）
- 需长期服用的药物
- 特殊性药物（如哮喘喷雾）

**携带要求**：

- 带足全行程的量，并多带3-5天余量
- 保留原包装和说明书
- 最好附英文处方或诊疗证明（以防海关查验）
- 液体药物需符合航空规定

### 3.3 特殊场景药品

| 场景 | 建议携带 |
|------|----------|
| 海岛/热带 | 防晒霜SPF50+、晒后修复、驱蚊液 |
| 高原 | 红景天（提前吃）、止痛药 |
| 雨季潮湿 | 防止皮肤问题的药膏 |
| 长途飞行 | 耳塞、眼罩、U型枕 |

## 4. 电子设备及周边

现代旅行离不开电子设备，但要精简。

### 4.1 基础设备

| 物品 | 说明 |
|------|------|
| 手机 | 地图、翻译、支付、通讯 |
| 充电器 | 原装或符合规格的 |
| 充电宝 | 随身携带，不托运，容量≤100Wh |

**充电宝规定**：

- 随身携带，不托运
- 容量不超过100Wh（约20000mAh）
- 标识清晰，无破损

### 4.2 转换插头与充电

**必备工具**：

```
□ 万能转换插头（最好带USB和Type-C口）
□ 轻便插线板（如果有多个设备需要充电）
□ 备用充电线 ×1-2（容易损坏）
□ 取卡针（换SIM卡用）
```

**插头类型查询**：

- 出发前查目的地插头类型
- 确认自己的设备是否兼容
- 最好一个转换插头解决所有问题

### 4.3 拍照设备

**按需携带**：

| 类型 | 说明 |
|------|------|
| 手机 | 日常记录足够 |
| 微单/单反 | 专业摄影爱好者 |
| 运动相机 | 水下、极限运动 |
| 无人机 | 需了解当地法规（部分国家禁飞） |

**配件**：

- 存储卡（多带一张备用）
- 手机支架/自拍杆
- 相机包（保护设备）

### 4.4 通讯设备

| 类型 | 说明 |
|------|------|
| 主手机 | 日常使用 |
| 备用手机/SIM卡手机 | 主手机出问题时的备用 |
| 当地电话卡/eSIM | 流量和通话 |

## 5. 个护与日用杂物

小物品大作用，提升旅行舒适度。

### 5.1 洗漱包

**基础配置**：

```
□ 牙具（牙刷、牙膏，部分酒店不提供）
□ 洗发水/沐浴露（小瓶装的，够用就好）
□ 护肤品（防晒霜、乳液）
□ 化妆包（女生按需）
□ 毛巾（部分青旅/民宿不提供）
□ 梳子/小镜子
```

**技巧**：

- 超过100ml的液体需托运
- 购买旅行装或用分装瓶
- 洗发水沐浴露在当地买也行

### 5.2 防晒与护肤

| 类型 | 说明 |
|------|------|
| 防晒霜 | SPF50+、PA+++，防晒黑还防晒伤 |
| 晒后修复 | 芦荟胶或修复面膜 |
| 润唇膏 | 带防晒值的更好 |
| 晒后修复 | 芦荟胶，晒伤后用 |

### 5.3 驱蚊防虫

| 目的地 | 建议 |
|--------|------|
| 热带/东南亚 | 驱蚊液、驱蚊手环（防止登革热） |
| 草原/野外 | 防蜱虫、长袖长裤 |
| 欧洲/日本 | 一般不太需要 |

### 5.4 提升舒适度的小物

**旅途必备**：

```
□ U型枕（长途飞机/火车救星）
□ 眼罩（遮光睡觉）
□ 隔音耳塞（飞机上、吵闹的住宿）
□ 折叠购物袋（购物后装东西）
□ 密封袋（装湿衣物、分类收纳）
□ 小锁（酒店柜子锁）
```

### 5.5 杂物清单

```
□ 指甲刀（需托运）
□ 小剪刀（需托运）
□ 针线包（衣服扣子掉了可以缝）
□ 牙线（保护牙齿）
□ 纸巾/湿巾
□ 漱口水（长途飞行后用）
□ 茶叶包/咖啡（酒店可能有，但可以自带喜欢的）
```

## 6. 离线数字包

网络不是随时都有，提前准备好离线资源。

### 6.1 离线地图

**Google Maps离线地图**：

1. 手机打开Google Maps
2. 搜索目的地名称或找到位置
3. 点击地点→右上角"保存"→"离线地图"
4. 自动下载当前区域

**Maps.me**：

- 下载特定国家的离线地图
- 无需网络也能导航
- 适合户外徒步

### 6.2 离线翻译

**Google翻译**：

- 提前下载语言包
- 拍照翻译、语音翻译、键盘输入
- 在没有网络时也能用

**其他翻译App**：

- 有道翻译官
- 微软翻译

### 6.3 娱乐内容

**打发交通时间**：

- 提前下载电子书（Kindle、微信读书）
- 下载喜欢的播客（喜马拉雅、小宇宙）
- 提前存好喜欢的音乐/视频
- 下载好想看但没时间看的剧

## 7. 打包技巧与原则

### 7.1 打包流程

1. **列出清单**：按类别写出所有需要带的东西
2. **分类打包**：证件、衣物、电子、药品、杂物分开放
3. **检查清单**：出发前对照清单检查是否遗漏
4. **称重**：称一下行李重量，避免超重

### 7.2 重量控制

**建议重量**：

- 随身包：不超过7-10kg（飞机要求）
- 托运行李：根据航空公司规定（23kg或30kg）
- 总重量：不超过自己能轻松提起的范围

**减重技巧**：

- 不带过量的衣物，当地可以买/洗
- 不带纸质书，用手机/iPad看电子书
- 护肤品用分装瓶，减少重量
- 不带可能用不到的东西

### 7.3 分装与收纳

**收纳工具**：

```
□ 收纳袋（衣物、鞋子、杂物分类）
□ 压缩袋（厚衣服可以压缩）
□ 洗漱包（防水）
□ 鞋袋（隔离鞋子）
```

**分类建议**：

- 衣物按类型分：内穿、外穿、睡衣
- 杂物按用途分：洗漱、电器、文件
- 最重的东西放底部
> **小贴士**：出发前把行李拍照，行李牌也拍一张。万一行李丢失，可以更快向航空公司说明。买行李牌写上联系方式。贵重物品（相机、电脑）随身带，不托运。液体超过100ml托运，不超100ml随身。
