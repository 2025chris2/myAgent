## 🗺️ 三、推荐游览路线

### 🚶 一日游经典路线
上午：秦始皇兵马俑博物馆 → 下午：华清池 → 骊山 → 晚上：大唐不夜城

### 🚶 两日游深度路线
- **第一天：** 钟楼 → 西安城墙骑行 → 碑林博物馆 → 回民街（美食）→ 大雁塔夜景
- **第二天：** 秦始皇兵马俑 → 华清池 → 大唐芙蓉园 → 大唐不夜城

### 🚶 三日全景路线
- **第一天：** 陕西历史博物馆 → 大雁塔 → 大唐不夜城
- **第二天：** 秦始皇兵马俑 → 华清池 → 骊山
- **第三天：** 华山一日游（早出晚归）

> 📌 **温馨提示：**
> - 兵马俑距离市区较远（约40公里），建议预留半天时间
> - 华山在渭南市华阴市，从西安北站坐高铁约30分钟可达
> - 西安地铁网络发达，大部分景点可乘地铁到达
> - 春秋季节（3-5月、9-11月）是最佳旅游时间
> - 各博物馆建议提前预约门票
